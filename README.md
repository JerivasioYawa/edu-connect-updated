# EduConnect

EduConnect is a tutoring platform for students, by students.<br>
Its main objective is to provide learners with affordable and free extra classes that will enable them to study at their full potential.<br>
The core features of the application are: Registration of students and tutors; Session bookings, acceptance, and reviews; and Tutor reviews.
